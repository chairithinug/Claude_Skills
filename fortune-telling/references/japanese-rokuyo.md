# Japanese Rokuyō (六曜)

The 6-day cycle is tied to the lunar calendar (resets each lunar month), so it can't be computed from the Gregorian date alone without a lookup. **Use this tradition only when today's Rokuyō can be confirmed; otherwise skip it and rely on Thai or Chinese.**

## The six values

When the value is known:

- **Taian (大安)** — most auspicious, all hours. Good for weddings, openings, anything beginning.
- **Butsumetsu (仏滅)** — most inauspicious. Avoid weddings, openings, signing contracts. The "Buddha's death" day in name; the day to lay low.
- **Tomobiki (友引)** — lucky except midday (around 11–13). Avoid funerals (the name literally means "pulling friends" — you don't want to pull anyone into death). Otherwise fine for celebrations.
- **Sensho (先勝)** — morning lucky, afternoon less so. "First wins" — get the important thing done before noon.
- **Senbu (先負)** — morning unlucky, afternoon lucky. The mirror of Sensho. Hold the important thing until after lunch.
- **Shakkō (赤口)** — only the noon hour (roughly 11–13) is lucky; the rest is rough. Tight window — pick carefully.

## How to use

Rokuyō is a strong cross-check tradition for *timing within a day*, not for choosing whether to do something at all. If the user is asking "should I sign this?", Thai/Chinese gives the day's overall character; Rokuyō (when known) refines the hour.

For "is today auspicious for [event]?", Taian and Butsumetsu give clean signals; the morning/afternoon split values give nuanced ones.

## When to skip

If today's Rokuyō hasn't been confirmed by lookup and a tool isn't available to compute it, skip this tradition entirely. Don't guess the value.
